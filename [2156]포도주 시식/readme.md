 (1) i-3번째까지 마신 포도주의 최댓값 + 1칸 건너뛰고 + i-1번째에 주어진 포도주 + i번째 주어진 포도주를 마시는 경우이다.
		=> 1칸 건너 뛴 이유는 연속 3잔 이상 마시게 되는 경우를 피하기 위함이다.
 (2) i-1번째까지 마신 포도주의 최댓값 + i번째 마시지 않음의 경우이다.
		=> 왜 아무것도 안마시는 경우가 최댓값을 만드는데 도움이 될 수 있는가 라고 의문이 들 수 있지만, 잘 생각해보면 포도주가 0, 1 등의 극소량 밖에 없는데 이를 마셔버리면 다음 마실 수 있는 포도주에 연속 3개 불가의 제약으로 작용할 수 있기 때문에 안마시는 경우도 고려해야 하는 것이다.
 (3) i-2번째까지 마신 포도주의 최댓값 + 1칸 건너뛰고 + i번째 주어진 포도주를 마시는 경우이다. 
		=> i-2번째 포도주가 최댓값을 만드는데 도움을 주지 않는 경우를 고려해 제외하는 것과 동시에 연속3개 선택을 자연스럽게 막았다.
